BioSphere
=========

Watch boids evolve their genome to improve their traits in a competitive environment.


===================
System Requirements
===================

* OpenCL (see installing OpenCL)

----------------
 Supported OS
----------------

* Window 7 or higher


=================
Installing OpenCL
=================

-------------
 AMD
-------------

1. Make sure you have the most recent divers
  - http://support.amd.com/en-us/download
  
2. Download APP SDK
  - http://developer.amd.com/tools-and-sdks/heterogeneous-computing/amd-accelerated-parallel-processing-app-sdk/downloads/#appsdkdownloads
  
-------------
 NVIDIA
-------------

1. Get the latest driver
  - http://www.nvidia.com/Download/index.aspx?lang=en-us
